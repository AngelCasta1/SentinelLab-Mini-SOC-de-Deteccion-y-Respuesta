# SentinelLab-Mini-SOC-de-Detecci-n-y-Respuesta
Laboratorio casero de ciberseguridad que recorre el ciclo completo de un SOC: centralización de logs, detección en red y endpoint (Snort, Suricata, YARA), correlación de eventos en un SIEM (Wazuh + reglas Sigma), enriquecimiento con threat intelligence (AbuseIPDB, OTX) y gestión de vulnerabilidades (OpenVAS + DefectDojo).
